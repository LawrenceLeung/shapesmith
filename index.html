I'm building Shapesmith to be a free and open source 3D modelling project. It's aim is to make designing 3D printable models accessible to anyone with a modern browser (that supports WebGL). Right now you can draw shapes using primitives, perform boolean operations, do transformations, and export your models to STL format. See the blog for details on recent changes.

Feedback would be very much appreciated!

Get it from github

You can download and install Shapesmith from github.

Instructions for doing so on MacOSX and Ubuntu are available, but should be similar on other Linux platforms.

Installing it locally will produce much faster results that trying it out online.

Try it

I've installed a recent version (0.2) on an Amazon EC2 instance for you to try.

It's not a very powerful machine, so performance is unfortunately not fantastic.

Have fun!
